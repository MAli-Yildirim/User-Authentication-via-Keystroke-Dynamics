User Authentication via Keystroke Dynamics

Hi,
My name is Mehmet Ali Yıldırım and this is my graduation project User Authentication by using biometric information gathered from keyboards. The main idea is collecting enough information from your keyboard to calculate your typing habits then create a model for your password.

To do that first of all you need to register and login into the website then download the software from the link below. After that you can train your model using the program. After training your password (For good results you have to increase your database at least 100-200) you can import it to your account.

At the end, you could see the classification between users whos passwords are same. This project is pre-alpha so there could be bugs or errors. Also you can checkout my project on: https://github.com/MAli-Yildirim/User-Authentication-via-Keystroke-Dynamics.git
